Welcome to the Source Repo for The Programmer's Guide to Apache Thrift

The Programmer's Guide to Apache Thrift is a new book from Manning Publications. The book is currently in early access (MEAP) and approximately 75% of the chapters are available electronically now. The Apache Thrift example code in this repository is covered in detail in the book. To acquire a copy of the source, simply clone this repository:

$ cd path/where/you/want/the/source
$ git clone https://github.com/RandyAbernethy/ThriftBook

Working with the Source

Apache Thrift is a cross language RPC framework. The source code in this repo covers and range of languages and each will have its own prerequisites, as described in the book. The code examples generally require Apache Thrift v0.9.2 at a minimum.
